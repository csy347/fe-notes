## 2.MAC安装

**homebrew安装**

1. 先安装homebrew http://brew.sh/
2. 在terminal下安装 `homebrew`
3. 通过homebrew安装node.js
`brew install node`
4. 其他软件也都可以通过homebrew安装
`brew install mongodb redis git`


**n模块安装**

切换版本或升级node可以安装 `n` 模块
``` bash
npm install n -g
```

直接输入 `n` 即可上下切换不同的版本

![](5.nodeinstall/n.gif)

> n ls
> n use xxx
